## Data Structure Per Tool
Each tool import includes:
1. Basic Information
   - Title, content, excerpt
   - Tagline and website
   - Featured status

2. ACF Fields
   - Tool use cases (multiple)
   - Difficulty level
   - Key features (6 items with icons)
   - Pricing model and summary
   - Comprehensive stats

3. Taxonomies
   - Tool categories
   - Creation types
   - User situations
